# E-Cerpen
Basic E-Cerpen Web App built with GO and Vue.js
## Features
- Home Page
- Register, Login
- Upload Cerpen
- Kumpulan Cerpen Author
- View Cerpen
- Edit/Delete Cerpen
- Admin Panel:
  - Dashboard
  - Manage Cerpen
  - Validasi Cerpen

## Technology Stack
- Front-End: Vue.js
- Back-End: Golang
- Data Base: Supabase(PostgreSQL)
- Deployment: Heroku
- UI Automation testing: Cypress
- API testing: Postman
- Kanban Manajemen Tool: Notion
- Repository: Github
- Automation Server(CI/CD) : CircleCI
